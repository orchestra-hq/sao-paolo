# Security

## Reporting a vulnerability

Please **do not** file public issues for security vulnerabilities.

- Use [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories) for this repository if enabled, **or**
- Contact the maintainers through a private channel they publish for security reports (for example a security email listed in the organisation profile).

Include enough detail to reproduce or assess the issue (affected versions, configuration, and steps where safe to share).

## Supported versions

Security fixes are applied to maintained release lines as determined by the maintainers. Use the latest patch release where possible.
